# Dataset

## Overview

This folder contains the dataset used throughout the Incremental DBSCAN project.

The dataset serves as the input for data preprocessing, clustering experiments, and performance evaluation.

## Contents

| File | Description |
|------|-------------|
| `data_moons.csv` | Synthetic two-dimensional dataset used for clustering experiments. |

## Dataset Description

The dataset consists of two numerical features representing the coordinates of each data point. It was generated to evaluate the effectiveness of density-based clustering algorithms on non-linearly separable data.

The same dataset is used to compare the performance of:

- Incremental DBSCAN
- Traditional DBSCAN
- K-Means

## Purpose

The dataset is used to:

- Perform clustering experiments
- Visualize clustering results
- Evaluate clustering quality
- Compare different clustering algorithms
