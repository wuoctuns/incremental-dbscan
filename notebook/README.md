# Notebook

## Overview

This folder contains the Jupyter Notebook used to implement, test, and evaluate the Incremental DBSCAN algorithm.

The notebook demonstrates the complete workflow of the project, from data loading to clustering visualization and performance evaluation.

## Contents

| File | Description |
|------|-------------|
| `Incremental_DBSCAN.ipynb` | Complete implementation of the Incremental DBSCAN project. |

## Main Components

The notebook includes:

- Importing required libraries
- Loading and preprocessing the dataset
- Implementing the Incremental DBSCAN algorithm
- Running traditional DBSCAN
- Running K-Means clustering
- Visualizing clustering results
- Comparing algorithm performance
- Evaluating clustering quality using multiple metrics

## Output

The notebook generates:

- Clustering visualizations
- Execution time comparison
- Performance evaluation metrics
